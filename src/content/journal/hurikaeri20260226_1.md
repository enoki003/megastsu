# 情報処理部と各種イベント
入学して、私は情報処理部へ入部した。１０月まで学校で使用可能な情報機器がなかったため（スマホは校内で使用禁止）、ITパスポートの勉強をしていた。
先輩はDockerやCTFサーバの構築などを行っている上、高度情報技術者試験も受けているほどであるため、憧れであった。



# IT資格について
夏休みに入ってしばらく経った時、本来次の年の３月に受験するITパスポートが今の自分でもできるように感じたため、半年早く受験することを決定した。勉強方法は単純で、ITパスポートの教科書の中身をひたすらノートにまとめたのち、過去問道場で演習をしまくるといった具合である。そして１０月に受験し、600点ちょっとでギリギリ合格することができた。当初周りから称賛されて嬉しかったことを覚えている。
次に基本情報に向けて学習を始めた。そして基本情報技術者は次の年の２月合格した。その他では全商のビジネス文書実務検定を受けた。３級を授業で受けて、１級は余裕と感じたため受けた。1回目は筆記を落としたが、２回目の受験で合格した。

# シンギュラリティバトルクエスト
情報処理部に入部して、最初は特になにもしていなかった。パソコンが届いていなかったため、友達と話すだけとなっていた。ある時、先輩からシンギュラリティバトルクエストへの参加のお誘いがあったため、参加した。この大会では５つの競技種目があり、私はその中のセキュリティ部門のCTF(Capture The Flag)という種目に参加した。それまで、自分はパソコンの技術力が高いと思っていたが、思っていたより難しい問題が多く苦戦をした。そして２次選考の結果、私のチームが決勝に行けることとなった。大会本番、出だしは上位に行けていたのだが、難しい問題が残っていき詰まってしまった。そして結果はちょうど半分くらいの順位で、決して良いものではなかった。はじめて情報に関する大会に参加して、現実を知れたのは当時なんでも知っていると思っていた私には良い経験だったと思う。

# パソコンを組み立てる
中学３年性の頃に買ってもらったノートPCが突然潰れてしまった。親戚に見てもらったところ、どうやらマザーボードが故障しているようだった。保証も入っていなかったため断念するしかなかった。
しかし、流石に情報科学科でパソコンがないのは良くないと考え、親と相談し、勉強も兼ねて自分自身でコンピュータの構成を考え総額７万円ほどでPCを作成した。これは大成功で、今も問題なく使えているし、グラフィックボードがないのにも関わらずApexを軽々プレイできるほどであった。

#　暗記用のウェブアプリ制作
私の学校のテストはほとんど暗記だった。だから、暗記をしやすいように、問題をあらかじめ作り設定し、その問題を自分で解いて成否を判断するアプリを作った。実は中学の時にUnityで似たようなアプリは作成した。今回は一段階レベルアップとして、いつでもどこでもできるようにしたくて、Webアプリを制作することに決めた。当時はまだHTMLとCSSとJavaScriptの基本的な部分しか知らなかったが、インターネットを駆使してなんとか作った。サーバーサイドをPHPで実装しており、XAMPPを使っているような伝統的なやり方をしていた。結果的に作成はできて、借りていたサーバにデプロイもした。しかし、作ったものの受けは悪く、あまり使われずに終わってしまった。

# 一年生のまとめ
一貫して行きたかった高校に行けたという反動で、色々なことに挑戦し、また大きく成長した１年であったと思った。また、この一年の終盤からは私の関心はゲームではなく、興味を見失っていたと思う。
基本的には成功し、自信をどんどんつけていた年だったと覆う。身についた技術はWeb開発の基礎、Cの基礎、セキュリティ関連の部分、情報とビジネスの基礎知識であった。
しかし、そんな中、情報の道を志したものの、何をすればいいのかがわからなくなっていた。特に作りたいものも無くなっていたこともあった。
